# Error

The `<Error>` React Component is responsible for relaying any error messages back to the user of an interface.

The `<Error>` Component can do this in two ways, by providing an error as a prop, great for when handling direct API responses.
Alternatively you can insert a message within the 'child(ren)' of the component.

The component also accepts a 'title' prop to help give a summary of the message, e.g. 'Failed API response'.

All this information can then be used to provide feedback to the user about the error.
